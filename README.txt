trade-calc

[機能概要]
・クリック証券の取引履歴CSVデータから各種情報を抽出する

[要環境]
・ruby 1.8.7
・sqlite3
・sqlite3-ruby

[機能]
・クリック証券の取引履歴CSVデータを読み込む
・読み込んだデータから以下の情報を出力
　・年間損益
　・月別損益
　・年間損益（銘柄単位）
　・年間売買手数料
　・月毎売買手数料
　・年間買方金利・貸株料
　・買方金利・貸株料（銘柄単位）

[使い方]
・取引履歴CSVをクリック証券からダウンロードしてくる
　click証券ログイン後→aマイページ→精算表→取引履歴CSV(約定日を"今年分")
　※現物・信用両方のデータが必要です(現引き、現渡データを計算するため)
・売買データを ./data配下に配置して、以下のコマンドを実行
 % ruby import.rb
・情報を出力する時は以下のコマンドを実行
 % ruby extract.rb
